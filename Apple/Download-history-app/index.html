下载旧版app（Fiddler版）

2020-12-15

俗话说的好，Fiddler 学的好，牢饭吃到饱。

说明

iTunes大概从版本12.7.2.60开始不再支持下载应用程序，可以下载低版本

  1. Https 解密的准备工作这边就不在重述（繁琐又基础）

  2. iTunse 配合 Fiddler 对软件进行版本号抓取

  3. 启动 iTunse，登录账号
  4. 在 iTunse Store 搜索 app
  5. 在 Fiddler 中搜索（如果搜索出不来就先下载一次）
    找到类似 https://p11-buy.itunes.apple.com/WebObjects/MZBuy.woa/wa/buyProduct
    保存 Response body（响应体来看版本号）

以上操作目的都是为了查看版本号
以下操作准备下载

  1. Fiddler 关键字断点 bpu MZBuy.woa 回车
  2. 在断点处 Inspectors > TextView
  3. 修改< string > [版本号] < /string >
  4. 点击 Run To Completion 就会继续下载了
