Product Key（产品密钥）

将Autodesk产品作为尖端产品或套件安装时，需要产品密钥。这种情况下，即使是相同版本的Autodesk产品，也需要不同的产品密钥才能区分不同的产品包。
Autodesk 2019的每个产品密钥都有一个大写字母“K”。同样，Autodesk 2017，就是“I”，Autodesk 2018，就是“J”，Autodesk 2020，将会是“L”,以此类推。

安装激活

2021 安装激活



1.Install Autodesk Products
1.安装 Autodesk 产品

2.Use as Serial 666-69696969, 667-98989898, 400-45454545 066-66666666 ..
or anything matching those templates
2.使用一个序列号 666-69696969, 667-98989898, 400-45454545 066-66666666 ..
或任何一个匹配版本的

3.Use a Product Key (from product keys 2021.html)
3.使用一个产品密钥（从 product keys 2021.htm）

4.Finish the installation & restart Autodesk Product
Choose Enter a Serial
4.完成安装和重启启动 Autodesk 产品
选择一个序列号输入

ACTIVATION : We suggest blocking outgoing traffic (faster and easier to activate)
激活：我们建议您阻止流量出站（更快更容易激活）

5.Before clicking on Activate
5.在点击激活之前
You have 2 options :
你有两个选择 :

a) Disable Your network Card, pull the network cable out or block with firewall
(this is just to disable online checks) it will tell you that an internet Connection is Required
simply click on close and click on activate again
a) 禁用你的网卡，拔掉你的网线或阻止防火墙
（目的是为了禁用联网检测）它会告诉你必须连接互联网

OR
或者

b) Click on Activate and it will do an online check, simply click
on close and click on activate again.
b) 单击激活，它将进行在线检查，只需单击
单击关闭，然后再次单击激活.

Choose option a or b.
选择 a 或 b 方案

6.Select I have an activation code from Autodesk
6.选择 我具有Autodesk 提供的激活码

7.Once at the activation screen:
start XFORCE Keygen with administrator rights
7.来到激活界面
以管理员身份运行 XFORCE Keygen

8.Click on Mem Patch (you should see successfully patched)
8.点击 “Patch”（此时你看到的应该是 successfully patched）

9.Copy the request code into the keygen and press generate
9.复制申请号到 keygen 中的 Request 点击 “Generate”

10.Now copy the activation code back to the activation screen and click Next
You have a fully registered autodesk product
10.现在复制 keygen 中的 Activation 回到激活界面，点击 “下一步”
您有一个完全注册的 Autodesk 产品

You may get this message while your outgoing traffic is blocked, or you are off internet
Your online activation request has returned an error (800c0005) which is not allowing your request to be processed using our automated systems.
just Click Close
当您的传出流量被阻止或您不在互联网上时，您可能会收到此消息
您的在线激活请求已返回错误（800c0005），该错误不允许使用我们的自动化系统处理您的请求
只需点击关闭

Notes:
注意：

1- Untick the Autodesk Desktop App, Autodesk Single Sing-On and Autodesk Genuine Service before pressing install button
1- 取消勾选 Autodesk Desktop App，Autodesk Single Sign-On和Autodesk Genuine Service 之前，请先点击”安装”

2- Make sure you are running the Keygen as administrator from your HDD and with UAC off on Windows 7/8/10
You can apply updates before or after activation
2- 确保您从磁盘中以管理员身份运行 Keygen，并且在 Windows 7/8/10上关闭了UAC权限
您可以在激活之前或之后应用更新

3- Change the Autodesk Licensing Service to v9.2.2.2501 (use ChangeLicensing VBScript)
3- 将 Autodesk Licensing Service 更改为 v9.2.2.2501（使用 ChangeLicensing VB脚本）

安装问题

安装失败请尝试删除以下内容

  • 用户配置文件/目录
    • HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\Shell Folders
    • HKEY_CURRENT_USER\Software\Microsoft\Windows\CurrentVersion\Explorer\User Shell Folders
  • 安装产生的文件
    • C:\Program Files\Autodesk
    • C:\Program Files (x86)\Autodesk
    • %UserName%\AppData\Local\Autodesk
    • %UserName%\AppData\Roaming\Autodesk
    • C:\ProgramData\Autodesk
    • C:\Program Files\Common Files\Autodesk Shared
    • C:\Program Files (x86)\Common Files\Autodesk Shared
  • 许可文件 ADSKFLEX_NUMBERS_TSF.data
    • 显示隐藏文件
    • Windows XP:
      • C:\Documents and Settings\All Users\Application Data\FLEXnet
    • Windows 7/8/10:
      • C:\ProgramData\FLEXnet
    • 单用户固定期限许可登录信息
      • %USERPROFILE%\AppData\Roaming\Autodesk\ADUT
      • C:\ProgramData\Autodesk\ADUT
      • C:\ProgramData\Autodesk\CLM (2017 产品)
    • PIT 文件, ProductInformation.pit– 包含2010版本以后所有产品信息
      • Windows 7/8/10: C:\ProgramData\Autodesk\Adlm
      • Windows XP: C:\Documents and Settings\All Users\Application Data\FLEXnet
    • 许可
      • C:\Program Files\Common Files\Autodesk Shared\AdLM
  • 残留注册表
    • HKEY_CLASSES_ROOT\Installer\Products
    • HKEY_CURRENT_USER\Software
    • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Installer\UserData\S-1-5-18\Products
    • HKEY_LOCAL_MACHINE\SOFTWARE\Classes\Installer\Products
    • HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall
