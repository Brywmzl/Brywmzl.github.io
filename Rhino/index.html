超级强大而轻量级的NURBS建模软件

Rhino（全称：Rhinoceros，又名：Rhino3D）是一款轻量级但非常强大的高级3D建模工具，硬件配置要求较低。不要忽视它，因为它包含了所有的NURBS（非均匀有理B样条）建模函数。事实上，许多用户经常使用它来执行建模，然后导出其他3D软件使用的高精度模型。

rc 是 release candidate 的缩写 发布候选的意思。
sr 是 stable release 的缩写 稳定版的意思。
