PdfGrabber - PDF转换工具

2019-09-18
  • 办公应用

PdfGrabber 是一个德国的程序 它将虚拟PDF打印机轻松地将PDF文档转换为所有常用格式。

Homepage
2baksa.net

WINDOWS (32bBIT) WINDOWS (64bBIT) APPLE MACOS

下载

官方下载


Word
将PDF转换为Word并使用Microsoft Office或其他软件进行编辑。

Excel
使用Excel文件（.xls）中的表格转换PDF文档，并在Microsoft Office中打开它们。

PowerPoint
将PDF文档转换为PowerPoint演示文稿文件（.ppt）。

AutoCAD / DXF
将带有矢量的PDF文件转换为CAD图形（DXF，AutoCAD，DWG）。

图像
将完整的PDF文档转换为图像（JPEG，BMP，PNG，Tiff，SVG）…

导出图像
导出并保存PDF中的所有图像（JPEG，BMP，PNG，Tiff，SVG）…

PDF要求
原则上，支持所有PDF文档，这些文档都是从原始文档创建的，例如Word，PageMaker，QuarkXpress等。
请注意，PdfGrabber并不完全支持从右到左的语言，例如阿拉伯语，波斯语，希伯来语。
