引导

  • 在 BIOS 当中，有“UEFI+GPT”和“Legacy+MBR”启动模式
  • Auto/Both：自动按照启动设备列表中的顺序启动，优先采用UEFI方式
  • UEFI+GPT：快速启动（可绕过BIOS校检）
  • Legacy+MBR：传统启动
  • 具体设置还是要看 BIOS

分区4K对齐那些事

  • 在对磁盘进行分区时，有一个很重要的注意事项，就是要将分区对齐，不对齐可能会造成磁盘性能的下降。尤其是固态硬盘SSD，基本上都要求4K对齐。

在Windows下管理UEFI启动项

UEFI+GPT 模式

  • 用 PE 中的 DiskGenius 创建 ESP 分区

新建ESP分区

Legacy+MBR 模式

  • 用 PE 中的 DiskGenius 创建 ESP 分区
  • 这是最传统的引导方式
  • 和 UEFI 比起来，理论上是慢了一拍
  • 这也是最简单粗暴的引导方式
