2021 安装说明

0.在安装之前，通过 Windows 阻止 Internet 访问（简单来说断网）
不管你是防火墙阻止 还是 拔掉网线都行
检查 .NET Framework 3.5和4.0 是否已经安装
如果还没安装 .NET Framework 3.5（包括 .NET 2.0）
请前往“控制面板”->“应用和功能”->“启用或关闭 Windows 功能”
勾选“ .NET Framework 3.5（包括 .NET 2.0）“

00.SolidWorks_Flexnet_Server 如果存在，请从中卸载
你需要以管理员身份运行“SolidWorks_Flexnet_Server\server_remove.bat”
等到“SolidWorks_Flexnet_Server”被删除之后
从计算机中删除 SolidWorks_Flexnet_Server 文件夹

1.运行“sw2021_network_serials_licensing.reg”并确认添加注册表

2.复制文件夹“SolidWorks_Flexnet_Server”到C:
以管理员身份运行“SolidWorks_Flexnet_Server\server_install.bat”
等到“SolidWorks Flexnet Server”完成安装和启动

3.安装 SolidWorks 2021 （如果需要包括 PDM Client（设计图纸管理系统的客户端））
请勿安装 SolidNetwork License Server (SNL)!
当询问许可证服务器定义输入时: 25734@localhost

3.1 如果出现“系统检查警告”窗口，请忽略它（点击“下一步”继续）

3.2 如果出现“在激活数据库中未找到 SolidWorks 序列号”警告，请忽略他（点击“确认”继续）

3.3 如果看不到要安装的SW产品的完整列表，点击“选择其他软件包”，然后勾选“选择此包装中未包含的产品”

3.4 选择要安装的产品

3.5 如果出现“Installation Manager无法确定当前订阅截止日期。您要重新激活许可证以更新此信息吗?”（点击“否”，然后“是否要稍后再做”点击“是”）

4.安装完成后，覆盖 SolidWorks 2021 程序文件夹
从“Program Files\SOLIDWORKS Corp” 和 “Program Files (x86)”文件夹中
复制到“Program Files\SOLIDWORKS Corp”文件夹中替换（默认安装的情况下）：

要从“Program Files (x86)”文件夹中替换（默认安装的情况下）：

5.运行“SolidSQUADLoaderEnabler.reg”并确认添加注册表

6.重新启动计算机！

7.运行 SolidWorks > 帮助 > SolidNetWork License Manager > 许可顺序

使用“上移”和“下移”按钮将高级产品放置在同名产品高于专业和标准产品

点击“应用”保存设置

单击“确定”关闭 SolidNetWork许可管理器

8.尽情享用吧~

2018 安装说明

  • 运行注册机（有管理员权限的系统，请以管理员身份运行）
  • 点击 1.Set Serial Number 设置序列号
  • 选择 Force Local Activation Serial Numbers 强制本地激活序列号
  • 点击 Accept Serial Numbers 接收序列号
  • 点击 Yes 继续
  • 序列号更改 点击 确认
  • 点击 2.Activate Licenses 激活许可
  • 选择 SolidWorks XXXX Products 选择版本
  • 点击 Activate Licenses 激活许可
  • 点击 Yes 继续

    部分或全部的序列号在这个系统需要网络浮动许可。
    你要生成这个许可文件吗？
    他可以用在现在的电脑也可以用在别的电脑。

  • 如果不要许可文件点击 No
  • 许可激活 点击 确认
  • 断网后，运行安装程序 Setup.exe
  • 下一步
  • 下一步
  • 此时如果没有连接网络或连不上 就点 取消
  • 有安装过的会提示移除选项
  • 下一步
  • 选择不同软件包或产品 可以显示出隐藏选项
  • 简体中文要在 SolidWorks 语言 里面选择
  • 展开 SOLIDWORKS 里面有很多用不到的勾可以去掉。。。
  • 点击 现在安装
  • 下一步（这个过程比较漫长。。。）
  • 安装完回到注册机
  • 点击 3.Copy Loaders 拷贝装载
  • 装载拷贝 点击 确认
  • 完成激活！
  • 如果打开软件出现这个界面 那就是 激活失败 或者是 许可被吊销了
  • 点击 Remove All Loaders 移除所有装载
  • 点击 Purge All Activations 清除所有激活
  • 然后在执行上面的 注册机操作
