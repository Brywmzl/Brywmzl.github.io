安装说明

2020 安装说明

  • 如果之前有安装过任何的 SIEMENS SPLM License Server，请将其卸载
    • 如果安装的是原版的 SPLM License Server，请使用“程序和功能”来删除它
    • 如果安装了 SSQ SPLM License Server ，请从 SPLM License Server 文件夹中，以管理员身份运行 “server_remove.bat”
  • 不要从 NX 安装向导中安装原版的 SPLM License Server ！！！
  • 安装或更新 SolidSQUAD 通用许可服务
  • 解压 “_SolidSQUAD_.7z”
  • 打开 “Server” 文件夹
  • 将 “Vendors” 文件夹，覆盖 “SolidSQUAD_License_Servers” 中的 “Vendors” 文件夹
  • 然后将 “SolidSQUAD_License_Servers” 文件夹，复制到任意一个磁盘的根目录下（例：C:/ D:/ 等）

  • 从 “SolidSQUAD_License_Servers” 文件夹中，以管理员身份运行 “install_or_update.bat”

  • 点击“Install NX”

  • 选择语言，点击“确定”

  • 点击“下一步”

  • 选择组件和安装位置，点击“下一步”

  • 在安装程序中，当询问许可证服务器端口和名称输入时：
    27800@localhost;28000@localhost;C:\ProgramData\Siemens\vistagy_SSQ.dat

  • 选择语言，点击“下一步”

  • 点击“安装”

  • 安装过程漫长，请耐心等待。。。

  • 点击“完成”

  • 复制 “Client” 文件夹中的所有文件夹，替换原来的 (默认位置 C:\Program Files\Siemens\NX1899)

  • 运行 “SolidSQUADLoaderEnabler.reg” 点击“是”，将信息添加到 Windows 注册表中

  • 重新启动计算机
  • 请享用！
